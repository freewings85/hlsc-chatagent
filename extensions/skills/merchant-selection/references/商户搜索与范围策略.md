# 商户搜索与范围策略

## 搜索方式

1. 按车主需求搜索商户，关注或选择（选择后默认关注）
   - 可选择具体商户 ID
   - 也可选择商户范围和类型
2. 搜索特定具体商户（老商户，按 ID）
3. 不同项目默认地理范围不同（如保险项目为全城范围）

## 范围策略

范围策略由 search_nearby_shops tool 的入参承载，skill 不硬编码：
- 项目类型 → 默认搜索距离（由 tool 返回）
- 保修状态 → 影响推荐策略（由 tool 返回）
- 位置信息 → 通过 geocode_location 获取

## 注意

- 搜索结果的排序和推荐策略由 tool 返回，skill 负责向用户解释推荐结果
- 如果搜索结果不满意，可调整搜索条件重新搜索
