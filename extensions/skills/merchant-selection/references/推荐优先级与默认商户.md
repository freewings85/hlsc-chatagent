# 推荐优先级与默认商户

## 推荐优先级

在符合车主对商户需求条件的前提下，推荐商户时综合考虑：
- 车主消费偏好（T2）
- 项目特点
- 已建关系商户（已预订、已关注、已推荐老商户）
- 本次特别要求
- 好评度
- 平台官方评价

推荐优先级的具体规则由 search_nearby_shops tool 的入参和返回策略承载，
skill 负责向用户解释推荐结果、处理不满意的情况。

参考规则（最终以 tool 返回为准）：
- 未出保车辆的保养维修项目 → 优先推荐整车厂授权店（4S）
- 已出保车辆，特别是 6 年以上 → 优先推荐 4S 以外服务商户
- 已出保车辆的保养和轻度服务（洗车美容等） → 一般 3-5 公里就近推荐，优先老服务商户
- 保险项目 → 全城范围

## 管家服务默认商户

管家服务项目 → 自动选择默认商户
