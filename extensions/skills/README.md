# Skills 设计说明

## 产品定位

帮助用户制定省钱方案并完成下单服务。

## 省钱方式

当前平台支持的省钱方式：

1. **平台优惠方式** — 零部件九折、优惠券等平台提供的优惠，需确认项目后判断适用性
2. **保险竞价** — 保险类项目基本都能走竞价 PK，多家商户报价竞争
3. **商户自有优惠** — 部分商户有折扣、满减、会员价等活动，需按条件查找

用户不关心优惠时，跳过省钱流程直接进入执行。

## Skill 设计原则

1. Skill 提供的是 **LLM 自身不具备的业务知识**（平台规则、省钱方法、预订流程等）
2. LLM 自身能做的事（理解用户意图、组织语言、调用工具）不需要 skill 教
3. 每个 skill 应该围绕"帮用户省钱"这个核心目标

## 当前 Skills

### S1（体验模式）
- **saving-methods** — 3 种省钱方式的介绍和适用场景
- **platform-intro** — 话痨平台身份、能力、九折机制

### S2（下单模式）
- **saving-playbook** — 省钱下单全流程剧本（5 阶段 + 7 个 reference）
- **booking-execution** — 预订执行流程（标准/委托/竞标 3 种模式）
