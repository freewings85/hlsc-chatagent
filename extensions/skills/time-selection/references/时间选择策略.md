# 时间选择策略

## 第一步：实时 vs 可预约

### 实时/救援类
- 救援类项目（抛锚、爆胎等） → 实时，省时最优先
- 部分救援项目即使商户休息也需要有办法触达和反馈车主
- 救援类项目的消费偏好自动偏向 E 档

### 可预约类
- 非紧急项目 → 可预约

## 第二步：可预约服务的时间建议

提醒车主预订时间点的注意事项：
- 避开旺季、节假日前几天
- 避开商户休息时间
- 部分优惠券可能需要提前一定时间预订（具体由预订校验 tool 只读返回 eligibility）

## 第三步：执行阶段时间注意事项

对于可预订的项目，执行预订方案时需注意：
- 如果需要在线联系商户，则需要注意商户的作息时间，可能联系不上
- 如果是旺季，商户的在线回复可能会慢

## 考量维度

- **紧急程度**：抛锚、爆胎等 → 立即；常规保养 → 可安排
- **淡旺季**：春节前/长假/周末 → 旺季排队久；工作日 → 淡季
- **商户排期**：热门商户可能需要提前预约
- **车主日程**：工作日/周末偏好、可等待时间
