# 潜在需求引导

车主因缺乏专业知识而不清楚自己需要什么时的处理方法。

## 场景

- 车主未提具体需求，但车龄/里程已到某些项目的建议时间
- 车主提了一个项目，但根据车况可能还有其他应关注的项目

## 识别方法

1. 根据车型、车龄、里程，结合车辆生命周期保养清单
2. 主动沟通优先级高、应被关注的项目
3. 委托项目推荐 subagent 获取推荐
4. 引导保养维修的管家服务

## 注意

- 不要一次推荐太多项目，按优先级排序
- 先给价值再推荐："根据您的车已经跑了6万公里，刹车片可能需要检查一下"
- 尊重车主意愿，推荐但不强推
