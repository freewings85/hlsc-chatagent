# 事故与现场检查处理

## 无法远程确认的项目

大量维修保养或改装加装的项目，即便车主能描述一些需求，也无法准确判断本次要预订的项目时，
建议车主需要到商户现场检查和确认。

当项目匹配相关 tool 返回结果表明该项目需要现场确认时：

处理方法：
1. 告知车主该项目需要到店检查才能确认
2. 先筛选一家商户，做现场项目检查和项目导入系统
3. 如果有初步判断，告知可能性和大致费用范围
4. 检查后再回来确认最终项目

注意：
- 不要在无法确认的情况下给出确定性报价
- 检查本身可能有费用，提前告知

## 事故碰撞——自费不走保险

场景：车辆碰撞事故，车主选择自费维修不走保险理赔。

处理方法：
1. 了解事故基本情况（碰撞部位、严重程度）
2. 如车主说不清具体项目：描述 + 上传图片即可
3. 需确认是自己的车还是别人的车（都需要 VIN）
4. 引导进入后续流程

## 事故碰撞——走保险理赔

场景：车辆碰撞事故，车主需要走保险理赔。

处理方法：
1. 无论车主能否说清项目：描述 + 上传图片即可
2. 保险理赔流程可能影响商户选择
3. 涉及安全部件（碰撞后的气囊、转向系统），必须强调到店检查
