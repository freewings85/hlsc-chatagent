# 管家服务项目提取

## 管家服务项目提取（T3.5）

如果车主有"管家服务"，项目来源有三个：
1. 自动从"下次需要做的项目清单"中提取
2. 加上车辆生命周期项目清单中刚性需要做的项目
3. 以及到期提醒的项目

综合以上三个来源，确认本次的具体预订项目。

处理方法：
1. 从 USER.MD 或历史记录中获取管家服务计划
2. 提取"下次需要做的项目清单"
3. 结合生命周期清单中的刚性项目和到期提醒
4. 综合形成本次预订项目候选列表
5. 与车主沟通确认本次要做哪些项目

注意：
- 管家服务的项目清单可能包含多个项目，需要和车主确认优先级
- 如果管家服务关联了默认商户，后续 T5 可直接使用
