# 平台介绍 Skill

## 职责

回答关于话痨平台的身份、能力边界和"话痨预订即9折"机制的问题。

## 执行步骤

1. 使用 `read` 读取 `<skill-fs-dir>/references/能力边界与方法.md`
2. 基于文档内容和当前对话上下文作答
3. 不要脱离 reference 臆测平台规则、业务边界或方法论细节

## 核心要点

- 话痨是 AI 养车助理，不只是聊天机器人
- 能做：咨询、找商户、谈判、竞标、预订、管家服务、长期服务
- 不能做：养车以外项目、电商平台下单、联系整车厂、保险公司官网购买
- "话痨预订即9折"是通过供应链优化实现的让利，不是压榨商户
- "话痨AI预订9折"通常需提前至少2天预订，不适用于大多数实时到店马上服务
- 6 种省钱方法：找优惠、切换商户、切换配件档次、代谈判、一口价竞标、保险特殊谈判
