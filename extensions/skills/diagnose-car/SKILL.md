# 车辆故障诊断

根据用户描述的故障症状或部件问题，从故障现象库检索匹配数据并分析诊断。

## 判据

用户描述了车辆的异常现象、故障症状，需要分析原因。

**典型话术：**
- "刹车有异响"
- "方向盘抖动"
- "发动机故障灯亮了"
- "过减速带咚咚响"

## 输入信息

从用户描述和上下文中提取以下信息：

- key_word: 车辆诊断关键词(故障症状、异常部件等)

## 流程

### Step 1：检索诊断

获取到 key_word 后，使用 bash 执行检索命令。

**重要**：`<skill-dir>` 替换为 Skill 工具返回的实际路径。所有脚本必须在 skill 目录下执行。

```bash
cd <skill-dir> && python scripts/search_project_by_keyword.py --keyword "刹车异响"
```

### Step 2：展示结果

将检索结果整理后展示给用户，包含诊断标题、内容描述及关联的推荐项目

## 返回格式

必须严格返回纯 JSON，不要包含任何额外文字、markdown标记或解释说明
