# T7-F 打包目标拆解预订

一个总目标需要分解成多个子预订项目分别执行。

## 流程

1. 分析总目标，拆解为多个独立的子预订项目
2. 每个子预订项目分别走 T7-B（标准）或 T7-C（竞标）流程
3. 按依赖关系和时间顺序排列执行计划
4. 需委托 execution-planner-agent subagent 进行复杂编排

## 注意

- 打包拆解的子项目之间可能有依赖关系（如先检查再维修）
- 每个子项目的 plan_mode 可能不同
- 拆解后需再次与车主确认整体方案
