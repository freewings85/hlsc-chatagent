# 省钱行情参考

本文件为行情数据参考，持续更新。用于帮助车主了解省钱的可能性边界。

## 渠道差异

- 4S 店 vs 平台推荐商户：降 30%-50%
- 不同品牌档次的配件差异：30%-70% 不等
- 保险返现差异：数百元不等

## 价格透明类项目

- 洗车、补胎等明码标价的小项目，价格很透明，差别不大

## 已是最低价的场景

- 部分项目（如保养和轮胎项目），特别是顶级品牌的保养（美嘉壳机油）和大牌轮胎更换
- 在快修连锁平台（如途虎、天猫养车）已是市面最低价
- 切到其它渠道最多价格相近，除非更换品牌，否则不大可能更低

## 不可实现的边界

结合车主的省钱目标和上述行情：
- 如目标太高无法实现，应直接反馈车主
- 如有些省钱方法话痨目前无法完成的，也应直接反馈
