# 谈判与竞标规则

## 方法 D：话痨代谈判

通常适用于"话痨AI预订9折"的项目，最终以 tool/API eligibility 为准。

特点：
- 可在车主有目标价或没有目标价的情况下进行
- 可在单个或多个商户（新商户、老商户均可）进行
- 进行一轮或多轮谈判
- 每一轮谈判的价格是可变的

→ 需委托 negotiation-bidding-agent subagent 执行

## 方法 E：一口价竞标

特点：
- 车主输入一口价的价格
- 多个服务商在线"竞标"/"抢单"
- 价格不变（与谈判的区别：谈判价格可变，竞标价格固定）
- 多轮推送 AI 生成的引导竞标抢单信息，提高成功率

→ 需委托 negotiation-bidding-agent subagent 执行

## 方法 F：保险特殊谈判

买保险这个特殊养车项目：
- 话痨代与商户谈判更多赠送或更多返现
- 一般可以谈下更好的返现优惠

→ 需委托 negotiation-bidding-agent subagent 执行
