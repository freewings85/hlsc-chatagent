# 配件档次切换规则

## 四个档次

| 档次 | 说明 | 价格区间 | 省心程度 |
|------|------|----------|----------|
| 原厂件 | 整车厂品牌配件 | 最高 | 最省心 |
| 配套国际 | 国际大牌（博世、曼牌等） | 较高 | 省心 |
| 国货精品 | 国内头部品牌 | 中等 | 有保障 |
| 杂牌 | 无品牌或不知名品牌 | 最低 | 风险较高 |

## 切换规则

- 逐级降档可降低成本（原厂 → 配套国际 → 国货精品）
- 不建议降到杂牌（涉及安全的配件绝不用杂牌）
- 车主消费偏好为"省心"时，不主动建议降档
- 某些配件（如机油滤芯）降档空间小，某些（如刹车片）降档空间大
- 需要提前确认车主的接受度
