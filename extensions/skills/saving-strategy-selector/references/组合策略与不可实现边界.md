# 组合策略与不可实现边界

## 方法组合规则

- 方法 A（找优惠）可与 B/C 叠加
- 方法 B（切换商户）和 C（切换配件档次）可叠加
- 方法 D（谈判）和 E（竞标）互斥——同一项目不能又谈判又竞标
- 方法 F（保险特殊谈判）仅适用于保险品类

## 不可实现时的处理

当以下情况出现时，应直接反馈车主，不硬凑方案：
- 省钱目标明显超出行情可能性（如"4S 店价格再降 80%"）
- 车主指定的渠道/方法话痨无法执行
- 不可预订渠道的价格仅供参考，不纳入执行方案

## 方法选择原则

1. 先检查方法 A（优惠），成本最低效果最直接
2. 再根据车主消费偏好（T2）判断 B/C 的接受度
3. 方法 D/E/F 需要 subagent 支持，当前如未实现则告知车主
4. 行情参考用于判断边界，不是独立的省钱方法
