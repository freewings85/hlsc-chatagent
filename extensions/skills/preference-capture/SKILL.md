# 消费偏好捕获 Skill（T2）

## 职责

识别并确认车主的消费偏好，映射为 A-E 五档之一，输出结构化偏好信息供后续任务使用。

## 执行步骤

1. 先阅读 `read <skill-fs-dir>/references/消费偏好维度与识别方法.md`
2. 优先从 USER.MD 获取已有偏好记录，直接映射为 A-E 档位
3. 如无记录，可直接向车主询问
4. 不同项目允许有局部偏移（如紧急救援可自动偏向 E 档）

## A-E 五档量化模型

| 档位 | 描述 | 比例 |
|------|------|------|
| A | 极致省钱 | 省钱 > 80% |
| B | 省钱为主兼顾省心 | 省钱:省心 = 60:40 |
| C | 省钱省心并重 | 50:50 |
| D | 省心为主兼顾省钱 | 省钱:省心 = 40:60 |
| E | 省心省时为主只要不做冤大头 | 省钱:省心/省时 = 20:80 |

## 完成标准

- 已确认 A-E 五档之一
- 偏好已记录到 context，供后续省钱方法选择和商户选择使用
