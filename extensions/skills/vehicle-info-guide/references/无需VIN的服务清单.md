# 无需 VIN 的服务清单

以下服务类型仅需 L1（品牌+车系）即可，不需要 VIN：

- 洗车
- 简单美容（打蜡、抛光）
- 通用咨询（养车技巧、行业信息）
- 违章查询
- 年检提醒（只需知道车辆注册日期）

以下服务需要 L2（品牌+车系+年款+排量）：

- 常规保养（机油+机滤）
- 标准配件更换（雨刮、空调滤芯等通用件）
- 一般报价比较

以下服务必须 L3（VIN）：

- 精准报价（含配件型号匹配）
- 配件兼容性校验
- 高风险维修决策（发动机、变速箱相关）
- 生命周期项目推荐

以下服务除车型外还需轮胎规格（如 225/45R17）：

- 换胎（需匹配轮胎型号）
- 补胎（部分场景需确认规格）
- 轮胎检测

轮胎规格获取方式：
- 引导车主查看轮胎侧面标记
- 引导车主拍照完整轮胎照片，调用 tire_image_recognize tool 自动识别
- "是否需要 tire_spec"由 match_project / get_project_price API 返回 missing_fields 判断
