# 方案制定方法论

在 Planning 阶段，需要组合省钱方法，为车主制定可落地的养车方案。

## 执行步骤

1. 使用 `read` 工具读取 `<skill-fs-dir>/references/制定方案与策略执行方法论.md`，获取完整的方案制定流程。
2. 基于文档内容和当前对话上下文作答。
3. 不要脱离 reference 臆测平台规则、业务边界或方法论细节。

## 方案制定流程

1. 根据 VIN/车型车况，结合车主偏好，梳理近期所有可能的需求项目
2. 针对每个项目，组合使用省钱方法：
   - 更换服务商户（找更优的）
   - 调整配件品牌档次（原厂→配套国际→国货精品）
   - 与商户价格谈判（新老商户）
   - 优惠谈判（特别是"话痨预订即9折"）
   - 多商户竞价
3. 给出可比较的 A/B 方案，让车主选择
4. 如果是长周期服务需求，制定年度级多次服务方案

## 商户限制规则

话痨不能预订的渠道（整车厂、途虎/天猫/京东养车官网、保险公司官网等），不应纳入执行方案中的商户。仅可在咨询对话中客观分析。
