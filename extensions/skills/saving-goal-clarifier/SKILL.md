# 省钱目标澄清 Skill（T3 后半）

## 职责

把"想省钱"变成可执行的目标，通过 3 个步骤逐步确认。

## 执行步骤

1. 先阅读与当前问题相关的 reference：
   - `read <skill-fs-dir>/references/省钱目标确认方法.md`
   - `read <skill-fs-dir>/references/对比基准类型与处理方式.md`
2. 按 3 步与车主确认省钱目标
3. 如车主目标不清晰，可调用报价查询 tool 先试探行情再反向确认
4. 输出明确的省钱目标供后续省钱方法选择使用

## Tools

- `get_project_price`：查询项目在附近门店的报价（试探市场行情）

## 3 步目标确认

### 步骤 1：了解车主原有服务商户类型
- 4S 店？途虎天猫养车等线上平台？电商买件找店装？周边小店？其它？
- 车主不清楚或不愿说时，不强制

### 步骤 2：了解车主对本次服务的商户基本要求
- 车主不清楚或不愿说时，不强制

### 步骤 3：了解车主的省钱目标期望值
- 大概比例？
- 直接金额区间？
- 有优惠就行，取决于有哪些优惠？
- 目标不清晰，需要先报价或给出行情价，再择优选择？
- 有个心理价位，能满足就行？

## 完成标准

- 省钱目标已量化（金额/比例/心理价位/有优惠就行），或车主明确表示"不在意价格"
- 原有商户类型已了解（如车主愿意说）
- 商户基本要求已了解（如车主愿意说）
