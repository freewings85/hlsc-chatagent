# 省钱目标确认方法

## 3 步确认流程

### 步骤 1：了解车主原有服务商户类型

与车主沟通中确认其原有的服务商户类型：
- 4S 店？
- 途虎天猫养车等线上平台？
- 电商大平台买件找店安装？
- 线下的周边较近的服务商店？
- 其它？

注意：车主不清楚或不愿说时，不强制确认。

### 步骤 2：了解车主对本次服务的商户基本要求

询问车主对本次服务希望的商户基本要求。
注意：车主不清楚或不愿说时，不强制确认。

### 步骤 3：了解车主对本次养车的省钱目标期望值

省钱目标类型：
1. **明确比例**：大概省百分之多少
2. **明确金额区间**：希望控制在某个价格范围内
3. **有优惠就行**：取决于有哪些优惠
4. **先看行情再定**：目标不清晰，需要先报价或给出行情价，再择优选择
5. **心理价位**：有个心理价位，能满足就行

要结合项目本身的特点和有无特殊要求，以及车主的消费偏好来确认。

## 目标不清晰时的处理

如果车主对省钱目标不清晰，可以：
1. 调用报价查询 tool 查询行情价格
2. 基于行情价格给车主一个参考区间
3. 让车主在了解行情后确认目标
