# 对比基准类型与处理方式

## 对比基准类型

车主的省钱目标通常有一个"和谁比"的基准：

| 基准类型 | 说明 | 处理方式 |
|----------|------|----------|
| 和 4S 店比 | 最常见，车主习惯在 4S 店做 | 查询 4S 店报价作为对比基准 |
| 和上次比 | 上次在某店做过 | 从历史记录获取上次价格 |
| 和某个价格比 | 车主心里有个数 | 直接以该价格为基准 |
| 和线上平台比 | 途虎天猫等 | 客观告知平台差异 |
| 没有基准 | 车主不知道多少钱合理 | 先试探行情再定 |
| 原有商户类型 | 车主习惯某类商户 | 了解原有类型后对比推荐 |

## 比较维度

- **总价**：最常用
- **工时费**：部分车主关注
- **配件费**：配件降档时关注
- **综合成本**：包含交通、时间等隐性成本

## 处理原则

- 对比基准必须是可获取的，否则建议先查行情
- 不可预订渠道的价格仅作参考，不纳入执行方案
