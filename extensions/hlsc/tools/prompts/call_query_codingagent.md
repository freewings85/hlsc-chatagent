说明：
当你已经明确要查什么、按什么条件查、需要返回什么结果时，调用这个工具，把查询任务交给 QueryCodingAgent。

When to use:
- 需要做查询、筛选、排序、聚合或结果整理。
- 主流程需要先拿到查询结果，再继续做解释、推荐、比较或确认。

Usage notes:
- 传给它的 `query` 要简洁、完整、可执行，建议用结构化 Markdown 描述，不要写成散文。
- 至少说明：查询目标、已知条件、必须满足的条件、优先考虑的条件、希望返回的结果。
- 它只负责查询和结果整理，不负责业务推荐、用户沟通或最终决策。
- 不要把含糊的问题直接交给它；主流程应先把条件补清楚，再调用。

Example usage:
```md
# 查询目标
搜索洗车商户

# 已知条件
- 位置：徐汇滨江
- 范围：10 公里内

# 必须满足
- 支持洗车

# 优先条件
- 评分 4.5 以上
- 距离更近优先

# 返回结果
- 返回 5 家商户
- 每家包含名称、距离、评分
```
