根据用户提到的养车需求关键词，匹配到对应的项目，返回 project_id 和项目名称。当用户提到服务项目名称（如"换机油"、"轮胎"、"保养"）时调用此工具。

返回：
- project_name_keyword — 用户原始关键词
- projects — 匹配到的项目列表，每个项目包含 project_id 和 project_name

根据匹配结果处理：
- 匹配到 1 个 → 直接使用该项目
- 匹配到多个且能从用户原话推断最可能的 → 直接选最匹配的那个
- 匹配到多个且无法推断 → 简要列出让用户选
- 无匹配 → 追问用户具体想做什么项目

IMPORTANT: project_id 必须来自本工具的返回结果，不可编造。
