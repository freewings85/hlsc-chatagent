Description:
根据用户提到的养车需求关键词，匹配到对应的项目分类和具体服务项目。

Usage notes:
- 返回三组匹配结果，置信度依次递减：
  - exact：名称精确匹配，置信度最高
  - fuzzy：名称包含关键词，置信度较高
  - rag：语义相似，可能有噪音，需结合用户原话判断相关性
- leaf=true 是具体服务项目（可用于搜索），leaf=false 是分类节点
- path 表示项目在分类树中的位置（如 "洗车美容 > 洗车"）
- 返回多个结果时，看 path 判断：同一 path 前缀下的直接全部使用，不追问；跨不同 path 的简要列出方向让用户确认
- 有 exact 匹配时优先使用
- project_id 必须来自本工具的返回结果，不可编造
